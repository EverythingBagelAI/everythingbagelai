# EverythingBagel AI

A comprehensive directory of AI applications and automation workflows with specialized chat interfaces.

## Features

- AI Applications Directory
- Automation Workflows
- Specialized Chat Interfaces
  - Code Assistant
  - Automation Expert
  - PRD Companion
  - AI Applications Guide

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase
- OpenAI

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   - Copy `.env.local.example` to `.env.local`
   - Fill in the required environment variables
4. Run the development server:
   ```bash
   npm run dev
   ```
